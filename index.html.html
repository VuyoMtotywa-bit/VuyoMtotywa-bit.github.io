📐 VB.Net · OOP
Cylinder & Cone Volume Calculator
github.com/VuyoMtotywa-bit/Database-Projects
What it is

A Windows desktop application that calculates the volume of geometric shapes — specifically cylinders and cones — based on measurements entered by the user. Built using Object-Oriented Programming principles, meaning each shape is structured as its own self-contained unit of code with its own properties and calculation logic.

  • User enters the radius and height of a cylinder or cone and the application instantly calculates and displays the correct volume
  • Each shape (Cylinder, Cone) is coded as a separate class with its own properties — this is the OOP principle of encapsulation in action
  • Input validation ensures users cannot enter letters, negative numbers, or blank values — the application handles these gracefully
  • Clean Windows Forms interface designed for ease of use — demonstrates understanding of user-facing application design
  • Demonstrates the difference between OOP (object-oriented) and POP (procedural) approaches to solving the same problem

⚙️ OOP · Fundamentals
Object-Oriented Programming — Shape Calculations
github.com/VuyoMtotywa-bit/Object-Oriented-Programming
What it is

A focused programming project demonstrating the four core principles of Object-Oriented Programming through shape and geometry calculations. This is important because OOP is the foundation of almost every modern programming language and framework in professional software development.

  • Encapsulation — each shape's data and calculation methods are kept inside its own class, not scattered across the code
  • Abstraction — complex calculation logic is hidden behind simple method calls, so you call CalculateArea() without needing to know how it works internally
  • Inheritance — shared shape properties are defined once in a parent class and reused by child classes (e.g., Circle and Rectangle both inherit from a base Shape class)
  • Demonstrates clean, readable code structure — an important quality for any professional development environment
